# react-sql

Starter to create a web app with:
 - Client (React)
 - Server (Node.js)
 - SQL Database (SQLite) + ORM (Objection.js)

You can also
 - Use another database such as PostgreSQL
 - Use another ORM such as TypeORM
 - Use TypeScript
 - Use another view library such as Vue.js
