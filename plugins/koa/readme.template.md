# `@reframe/koa`

Use Reframe with Koa.

### Usage

Add `@reframe/koa` to your `reframe.config.js`:

~~~js
!INLINE ./example/reframe.config.js --hide-source-path
~~~

Then eject the server code:

~~~js
$ reframe eject server
~~~
